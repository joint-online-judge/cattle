# cattle

The new generation of JOJ Frontend. Cattle are commonly raised as livestock for M(obx)eat and M(aerial U)Ilk, and also used as R(eact)iding animals.

## Requirements

* node.js/npm

  ## Setup

  ```
  $ npm install
  ```

  ## Running

  ```
  $ npm start
  ```

  ## Build

  ```
  $ npm run build
  ```

  ## Code Format

  ```
  $ npm run prettier
  ```
