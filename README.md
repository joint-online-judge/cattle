# cattle

The new generation of JOJ Frontend. Cattle are commonly raised as livestock for M(obx)eat and M(aterial U)Ilk, and also used as R(eact)iding animals.

## Requirements

* node.js/npm

## Setup

``` bash
npm install
```

## Running

``` bash
npm start
```

## Build

``` bash
npm run build
```

## Code Format

``` bash
npm run prettier
```
